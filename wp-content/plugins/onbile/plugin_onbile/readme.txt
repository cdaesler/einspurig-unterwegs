== Description ==

Mobile version of your wordpress. 

with Onbile your wordpress have a  Mobile Version with this plugin your users can visit your wordpress with theirs smartphones and visualize all content in a friendly way.
Feateures
- Multiple themes.

- Multiple configuration.

- Total customization.

- Change colors, texts, icons, background, etc.

- Your users can comment.

- compatible with all smarts phones ( Android, iPhone, blackberry, nokia, opera mini, etc.. )

System recommandations:
WordPress 2.7 or higher.
PHP version 4.4.2 or higher.
MySQL version 4.1 or higher.


== Installation ==

To install the newsletter plugin:

Don't forget to read how to install, in the documentation section. http://www.wp-newsletter.com

1. Upload `Onbile` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress ( activate Onbile Plugin ).
3. Follow the steps.
	3.1 Create an account in onbile.
	3.2 Select an template.
	3.3 Personalize theme.
	3.4 Active account.
	3.5 Paste account code in your wordpress.



== Frequently Asked Questions ==

= A question that someone might have =

= Where is my theme files? =
  your mobile version files are in www.onbile.com, don't worry about that, you  should  personalize your colors only , texts, and icons.

= Where i can change the texts of my mobile website? 
  if you have selected some theme with independent text on your wordpress and you wish to change any text, page or button text in few steps, log in www.onbile.com and you can to do this easy.

= my users have all features like a wordpress? =
  Yes through our api, your user can have all features and they can comment, search, and see all content like on your wordpress theme.

  **spanish**

= Donde están los archivos de mi tema? =
  Los archivos están en la web www.onbile.com, no te preocupes por eso tu solo debes personalizar la plantilla, colores, textos e iconos.

= Donde cambio los textos de mi plantilla movil? =
  Si has seleccionado alguna plantilla con texto independiente a tu wordpress y deseas cambiar algún testo de pagina o botón simplemente entra en www.onbile.com con tu usuario y lo podrás hacer de una manera muy cómoda y fácil.

= Van a redireccionar mi pagina fuera de mi dominio? =
  No tus usuarios no notaran diferencia entre tu versión móvil y tu versión clásica ya que las 2 correrán bajo tu url no serán redireccionados.

= Pueden comentar, buscar  y hacer todas las gestiones de wordpress? =
  Si mediante la api del propio pulgón tus usuarios comentaran, podrán realizar búsquedas, podrán ver todo el contenido de tu blog paginado etc...
